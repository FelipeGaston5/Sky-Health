ID: RF-01
Nome: Acesso por Perfil de Usuário
Tipo: Funcional
Prioridade: Alta

Descrição:
    O sistema deve permitir autenticação e acesso diferenciado para diferentes perfis de usuário (gestores, médicos, enfermeiros e pacientes), garantindo que cada usuário visualize apenas as funcionalidades e informações permitidas para seu perfil.

Pré-requisitos:
    Nenhum

Fluxo de Execução:

    Usuário acessa a tela de login do sistema;

    Insere credenciais (usuário e senha);

    Sistema valida as credenciais no banco de dados;

    Sistema identifica o perfil do usuário autenticado;

    Sistema redireciona para a interface específica do perfil;

    Sistema carrega apenas as funcionalidades permitidas;

Resultado Esperado:
    Usuário é autenticado com sucesso e visualiza somente as funções e informações permitidas para seu perfil específico.

Perfis Definidos:

    Gestor: Acesso completo a relatórios e dados estratégicos;

    Médico: Acesso a prontuários, prescrição digital e histórico de pacientes;

    Enfermeiro: Acesso a registros de enfermagem e dados básicos do paciente;

    Paciente: Acesso ao próprio histórico e agendamentos;
