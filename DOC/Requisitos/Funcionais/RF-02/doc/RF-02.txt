ID: RF-02
Nome: Atualização em Tempo Real
Tipo: Funcional
Prioridade: Alta

Descrição:
    O sistema deve refletir imediatamente qualquer alteração nos registros médicos, garantindo que todas as unidades de saúde conectadas tenham acesso às informações atualizadas simultaneamente.

Pré-requisitos:
    RF-00 (Cadastro Único de Paciente)

Fluxo de Execução:

    Usuário autorizado edita o registro de um paciente existente

    Sistema permite a modificação dos campos permitidos

    Usuário salva as alterações

    Sistema atualiza o registro no banco de dados central

    Sistema notifica todas as instâncias conectadas sobre a atualização

    Outros usuários visualizam as alterações imediatamente

Resultado Esperado:
    Alterações nos registros médicos aparecem imediatamente em todos os módulos e terminais conectados ao sistema, garantindo consistência das informações.
